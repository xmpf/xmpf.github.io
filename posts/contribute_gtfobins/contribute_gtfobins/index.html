Contribute to GTFOBins

Introduction

I decided to write a guide, in order to help people who want to contribute to GTFOBins project.

manpages

A man page is a form of documentation common in Unix (e.g: $ man 2 open).

Google it

In order to find binaries that might allow us to read the contents of a given file, I will take advantage of Google’s capabilities, in order to avoid grep-ing inside /usr/share/man/man[N] directories (found using manpath).

  • Google dork: site:man7.org/linux/man-pages/man1/

How to contribute?

  1. Fork the project: https://github.com/GTFOBins/GTFOBins.github.io
  2. Create a branch: git switch -c <new_binary>
  3. Enter _gtfobins directory
  4. Copy an existing file to use as a template
  5. Commit changes and push
  6. Create a pull request and wait for the changes to be reviewed

Note: The available functionalities documented, can be found at https://github.com/GTFOBins/GTFOBins.github.io/blob/master/_data/functions.yml
